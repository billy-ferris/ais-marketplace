## Accomplishments
- Every `/manage/*` route element is now wrapped in `RoleGuard`: `brands` and all `listings/*` require `[ADMIN, MANUFACTURER]`; `categories` and all `approvals/*` require `[ADMIN]`. Retailers now fall through to `ForbiddenPage` (defense-in-depth for T-02.4-12).
- `useUpload` consumes the new `{ url, fields, publicUrl, key }` presign response: builds a `FormData`, appends all signed fields first then the `file` last, POSTs with no manual `Content-Type`, and checks `uploadRes.ok` before returning `publicUrl` (mitigates T-02.4-13, WR-02).
- `apps/web` type-checks and full Vite build both pass.
