**categories.ts:** pgTable('categories') with fields: id (identity PK), name (varchar 255 not null), slug (varchar 255 not null unique), icon (varchar 100 optional -- stores Lucide icon name as string), displayOrder (integer not null default 0), createdAt, updatedAt, deletedAt.

**brand-listing-images.ts:** pgTable('brand_listing_images') with fields: id (identity PK), listingId (integer FK to brandListings.id not null), imageUrl (varchar 1024 not null), displayOrder (integer not null default 0), isPrimary (boolean not null default false), createdAt.
