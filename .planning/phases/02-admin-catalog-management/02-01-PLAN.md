**categories.ts:** pgTable('categories') with fields: id (identity PK), name (varchar 255 not null), slug (varchar 255 not null unique), icon (varchar 100 optional -- stores Lucide icon name as string), displayOrder (integer not null default 0), createdAt, updatedAt, deletedAt.

**brand-listings.ts:** Create listingStatusEnum via pgEnum('listing_status', ['draft', 'active', 'sold_out', 'archived']). pgTable('brand_listings') with fields: id (identity PK), name (varchar 255 not null), description (text optional), brandId (integer FK to brands.id not null), status (listingStatusEnum not null default 'draft'), createdAt, updatedAt, deletedAt.

**inventory-skus.ts:** pgTable('inventory_skus') with fields: id (identity PK), listingId (integer FK to brandListings.id not null), name (varchar 255 not null), description (text optional), upc (varchar 20 optional), size (varchar 100 optional), casePack (integer optional), price (numeric 10,2 not null), msrp (numeric 10,2 optional), quantity (integer not null default 0), expirationDate (date optional), imageUrl (varchar 1024 optional), createdAt, updatedAt, deletedAt.

**listing-categories.ts:** pgTable('listing_categories') join table with listingId (integer FK to brandListings.id not null), categoryId (integer FK to categories.id not null), composite primaryKey on both columns.

**brand-listing-images.ts:** pgTable('brand_listing_images') with fields: id (identity PK), listingId (integer FK to brandListings.id not null), imageUrl (varchar 1024 not null), displayOrder (integer not null default 0), isPrimary (boolean not null default false), createdAt.
