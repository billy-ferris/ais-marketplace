**createCategorySchema (3 tests):**
- "should accept valid category input" -- safeParse({ name: 'Test Category' }) success=true
- "should reject empty name" -- safeParse({ name: '' }) success=false
- "should accept optional icon and displayOrder" -- safeParse({ name: 'Cat', icon: 'star', displayOrder: 0 }) success=true AND safeParse({ name: 'Cat' }) success=true (without optionals)

**createListingSchema (5 tests):**
- "should accept valid listing input" -- safeParse({ name: 'Test Listing', brandId: 1 }) success=true
- "should reject empty name" -- safeParse({ name: '', brandId: 1 }) success=false
- "should default status to draft" -- parse({ name: 'Listing', brandId: 1 }) and assert result.status === 'draft'
- "should require brandId as positive integer" -- safeParse({ name: 'Listing', brandId: -1 }) success=false AND safeParse({ name: 'Listing' }) success=false

**createSkuSchema (6 tests):**
- "should accept valid SKU input" -- safeParse({ name: 'SKU 1', price: '12.99', msrp: '24.99', quantity: 10 }) success=true
- "should reject empty name" -- safeParse({ name: '', price: '12.99', msrp: '24.99', quantity: 10 }) success=false
- "should require price as decimal string" -- safeParse with price: 12.99 (number not string) success=false AND price: 'abc' success=false AND price: '12.99' success=true
- "should require quantity as non-negative integer" -- safeParse with quantity: -1 success=false AND quantity: 0 success=true
- "should require msrp as decimal string" -- safeParse with msrp: 'abc' success=false AND msrp: '24.99' success=true

**updateListingSchema (1 test):**
- "should make all fields optional (partial of create)" -- safeParse({}) success=true AND safeParse({ name: 'Updated' }) success=true
