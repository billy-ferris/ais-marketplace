
**Step 2: apps/web/src/lib/api.ts:**
- Export `API_BASE_URL` from `import.meta.env.VITE_API_BASE_URL || 'http://localhost:3001'`
- Export `apiFetch` helper that wraps fetch with base URL, JSON headers, and credentials: 'include'

**Step 3: apps/web/src/lib/constants.ts:**
- Define nav items per role as data structures:
  - Admin: `[{ label: 'Dashboard', icon: LayoutDashboard, path: '/' }, { label: 'Manage', icon: Package, path: '/manage' }, { label: 'Users', icon: Users, path: '/users' }]`
  - Manufacturer: `[{ label: 'Dashboard', icon: LayoutDashboard, path: '/' }, { label: 'Inventory', icon: Package, path: '/inventory' }, { label: 'Offers', icon: HandshakeIcon, path: '/offers' }]`

- Export function `getNavItemsForRole(role: UserRole)` returning the correct array

**Step 4: apps/web/src/hooks/useRole.ts:**
- Import `useUser` from `@clerk/react`
- Extract role from `user.publicMetadata.role` (typed as UserRole)
- Return `{ role, isAdmin, isManufacturer, isRetailer, isLoading }` with boolean helpers
- Handle case where role is undefined (default to null, show loading state)

**Step 5: apps/web/src/main.tsx:**
- Wrap app in `<ClerkProvider publishableKey={PUBLISHABLE_KEY} afterSignOutUrl="/">`
- Use `VITE_CLERK_PUBLISHABLE_KEY` from import.meta.env
- Render `<App />`

**Step 6: apps/web/src/App.tsx:**
- Use Clerk Core 3 `<Show>` component (NOT `<SignedIn>/<SignedOut>` — those are Core 2)

- `<Show when="signed-in">` renders `<AppShell />`

  - Clean, centered layout with Tailwind (e.g., flex column, items-center, min-h-screen, bg-gray-50)

**Step 8: apps/web/src/components/layout/AppShell.tsx:**
- Import `SidebarProvider`, `SidebarInset`, `SidebarTrigger` from shadcn/ui sidebar

  - Top bar with `<SidebarTrigger />` (hamburger button) and `<UserButton />` from Clerk

**Step 9: apps/web/src/components/layout/AppSidebar.tsx:**
- Use shadcn/ui Sidebar components: `<Sidebar>`, `<SidebarContent>`, `<SidebarHeader>`, `<SidebarFooter>`, `<SidebarMenu>`, `<SidebarMenuItem>`, `<SidebarMenuButton>`, `<SidebarGroup>`, `<SidebarGroupLabel>`
