  <action>
Create the @ais/shared package with all type definitions, Zod schemas, and constants that apps/api and apps/web will consume.

1. **packages/shared/package.json**:
   - `"name": "@ais/shared"`, `"version": "0.0.0"`, `"private": true`
   - `"exports"` field mapping: `"."` -> `"./src/index.ts"`, `"./types"` -> `"./src/types/index.ts"`, `"./schemas"` -> `"./src/schemas/index.ts"`, `"./constants"` -> `"./src/constants/index.ts"`
   - Dependencies: `zod` ^3.x
   - devDependencies: `typescript` ^5.x

2. **packages/shared/tsconfig.json** — Extends `../../tsconfig.base.json`, include `src/**/*.ts`

3. **Types (src/types/)**:
   - **user.ts**: Export `UserRole` const object `{ ADMIN: 'admin', MANUFACTURER: 'manufacturer', RETAILER: 'retailer' } as const` and `type UserRole = (typeof UserRole)[keyof typeof UserRole]`. Export `User` interface with: id (number), clerkId (string), email (string), firstName (string | null), lastName (string | null), role (UserRole), companyId (number | null), createdAt (Date), updatedAt (Date).
   - **company.ts**: Export `CompanyType` const object `{ MANUFACTURER: 'manufacturer', RETAILER: 'retailer' } as const` and type. Export `Company` interface with: id (number), name (string), type (CompanyType), marginPercentage (string — numeric stored as string from Drizzle), contactName (string), phone (string), street (string | null), city (string | null), state (string | null), zip (string | null), createdAt (Date), updatedAt (Date).
   - **api.ts**: Export `ApiResponse<T>` type `{ data: T }`, `ApiError` type `{ error: string; code?: string }`, `PaginatedResponse<T>` type extending ApiResponse with `{ total: number; page: number; limit: number }`.
   - **index.ts**: Re-export all from user, company, api.

4. **Schemas (src/schemas/)**:
   - **company.ts**: Zod schema `createCompanySchema` validating: name (string, min 1, max 255), type (enum manufacturer/retailer), marginPercentage (string, regex for decimal like "10.00", optional default "10.00"), contactName (string, min 1, max 255), phone (string, min 1, max 20), street (string optional), city (string optional), state (string, exactly 2 chars, optional), zip (string, max 10, optional). Export `updateCompanySchema` as partial of create. Export inferred types.
   - **user.ts**: Zod schema `createUserSchema` validating: email (string, email), firstName (string optional), lastName (string optional), role (enum admin/manufacturer/retailer), companyId (number optional). Export inferred types.
   - **index.ts**: Re-export all.

5. **Constants (src/constants/)**:
   - **roles.ts**: Re-export UserRole from types. Export `ROLE_LABELS` record mapping each role to display label ("Admin", "Manufacturer", "Retailer"). Export `ALL_ROLES` array of all role values.
   - **routes.ts**: Export `API_ROUTES` const object with route path strings: `HEALTH: '/api/health'`, `USERS: '/api/users'`, `COMPANIES: '/api/companies'`, `WEBHOOKS_CLERK: '/api/webhooks/clerk'`, `AUTH_ME: '/api/auth/me'`.
   - **index.ts**: Re-export all.

6. **src/index.ts**: Barrel export from types, schemas, and constants.
